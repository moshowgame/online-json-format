02

關於

輕量、安全、零干擾的瀏覽器端 JSON/BSON 工具箱。

特性

  • 一框三用：同一個輸入框支援「格式化 / 壓縮 / 提取元素」三種操作，按需切換。
  • 純前端：所有計算在瀏覽器內完成，您的數據從不離開本機。
  • 零追蹤：不集成任何分析或廣告腳本。
  • 一頁完成：所有功能並排在同一頁，無需切換視圖。
  • BSON 友好：支持 Extended JSON、Hex、Base64 三種輸入模式，自動識別 BSON 特殊類型。
  • 鍵盤友好：Ctrl+Enter 執行、Ctrl+K 清空、Ctrl+/ 切換主題。

使用流程

  1. 在工具區選擇「操作」：格式化 / 壓縮 / 提取；
  2. 粘貼 JSON 或 BSON 文本，根據操作配置對應選項（縮進、路徑、條件等）；
  3. 點擊「執行」按鈕或 Ctrl+Enter，結果顯示在右側輸出區；
  4. 提取模式下可點擊行查看完整路徑與父對象詳情，支援複製 JSON 數組或下載 CSV。
